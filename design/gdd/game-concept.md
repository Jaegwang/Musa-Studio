## Elevator Pitch

> 조선의 전설적인 장수 소야차가 되어, 전장의 피를 흡수해 힘으로 바꾸는 쿼터뷰 액션 ARPG.
> 적의 배치를 읽고, 위치를 선점하고, 쓰러뜨린 뒤 흘린 피를 빨아들여 더 강해져라 —
> 그래야 다음 보스를 쓰러뜨릴 수 있다.

## Core Fantasy

전장에서 가장 두려운 존재가 되는 것.

소야차는 적을 죽이는 것에 그치지 않는다. 쓰러진 적의 피를 빨아들여 자신의 힘으로 만든다.
전투가 끝난 자리에는 핏자국 하나 남지 않는다 — 전부 그의 것이 됐으니까.

플레이어는 여기서 두 가지 쾌감을 얻는다:
하나는 포지셔닝을 통해 전장을 완벽히 통제하는 전술가의 쾌감,
다른 하나는 흘린 피를 모두 흡수해 보스를 압도하는 순간의 폭발감.

이 조합을 다른 게임에서는 찾을 수 없다.

---

## Unique Hook

디아블로2 스타일의 이소메트릭 액션 ARPG, **AND ALSO** 전장의 피를 직접 흡수해
전략적 자원으로 비축했다가 결정적인 순간에 폭발시키는 혈기(血氣) 시스템.

소야차가 "조선의 뱀파이어"로 불린 역사적 전설이 게임의 핵심 메커닉 그 자체다.

## Visual Identity Anchor

**방향: "핏빛 조선의 밤"**

어둡고 무거운 조선 역사화 미학에 초자연적 긴장감을 더한다.
핵심 비주얼 룰: *모든 화면은 "이 장수는 살아있는 무기다"라는 느낌을 줘야 한다.*

**지지 원칙:**
- **혈흡수 비주얼**: 지면의 핏자국이 입자가 되어 공중을 가로질러 주인공에게 빨려 들어간다.
  붉은 기가 사라지는 순간이 가장 강렬한 연출 포인트다.
- **무거운 색감**: 검정, 심홍, 탁한 황금. 조선 갑옷의 질감 + 전장의 먼지. 화려하지 않고 위압적이다.
- **카메라**: 쿼터뷰 고정. 전장 전체가 보여야 포지셔닝 판단이 가능하다.

**디자인 테스트**: "이 비주얼이 소야차의 잔혹함과 위압감을 강화하는가?" → 그렇지 않으면 재검토.

## Player Experience Analysis (MDA Framework)

### Target Aesthetics (What the player FEELS)

| Aesthetic | Priority | How We Deliver It |
| ---- | ---- | ---- |
| **Sensation** (sensory pleasure) | 2 | 혈흡수 파티클, 히트스톱, 타격감 있는 사운드 |
| **Fantasy** (make-believe, role-playing) | 1 | 조선의 전설적 장수로서 전장을 지배하는 판타지 |
| **Narrative** (drama, story arc) | 4 | 소야차의 전설을 따라가는 3챕터 서사 |
| **Challenge** (obstacle course, mastery) | 1 | 포지셔닝 숙달, 혈기 타이밍 결정, 보스 패턴 |
| **Fellowship** (social connection) | N/A | 싱글플레이어 |
| **Discovery** (exploration, secrets) | 5 | 조선 역사/전설 기반 적과 세계관 |
| **Expression** (self-expression, creativity) | 3 | 무기 선택, 혈기 소비 전략 |
| **Submission** (relaxation, comfort zone) | N/A | 도전 중심 게임 |

### Key Dynamics (Emergent player behaviors)

- 플레이어는 자연스럽게 적을 한 곳에 몰아 처치하고 혈흔을 밀집시키는 플레이를 발전시킨다
- 보스전 직전에 가능한 한 많은 혈기를 비축하는 루틴이 생긴다
- 흡수 가능한 혈흔이 있을 때 무리하게 전투를 이어가다 죽는 "욕심 실패"를 반복하며 학습한다
- 적 유형별 혈기 효과를 파악한 후 우선 처치 순서가 바뀐다

### Core Mechanics (Systems we build)

1. **포지셔닝 전투** — 쿼터뷰 기반 실시간 액션. 방향 기술, AoE 범위 공격, 군중 제압. 유리한 위치를 선점하지 않으면 소모전이 된다.
2. **혈흡수 시스템** — 처치한 적이 혈흔을 남긴다. 혈흔 위에서 흡수 발동 시 피가 입자가 되어 주인공에게 흘러들어오며 혈기 충전. 흡수 중 짧은 취약 구간 발생.
3. **혈기(血氣) 전략 자원** — 충전된 혈기는 즉시 소비되지 않고 비축된다. 보스전, 엘리트 밀집 구간 등 결정적 순간에 방출해 능력 강화. "언제 쓰느냐"가 핵심 전략 결정.
4. **챕터 기반 적 설계** — 챕터마다 다른 적 유형(왜군/역적/요괴). 유형별 혈기 효과 상이. 적 구성을 읽고 처치 우선순위를 결정해야 최대 혈기를 확보할 수 있다.
5. **무기 시스템** — 카타나(근접 콤보, 포지셔닝 이동기)와 활(원거리, 혈흔 생성 위치 조정). 상황에 따라 전환.

## Player Motivation Profile

### Primary Psychological Needs Served

| Need | How This Game Satisfies It | Strength |
| ---- | ---- | ---- |
| **Autonomy** (freedom, meaningful choice) | 어떤 적을 먼저 처치할지, 언제 혈기를 쓸지 — 모든 전투가 플레이어의 판단 | Core |
| **Competence** (mastery, skill growth) | 포지셔닝 숙달 → 더 효율적인 혈기 수집 → 더 높은 보스 처치 → 명확한 성장 곡선 | Core |
| **Relatedness** (connection, belonging) | 소야차라는 실존 인물의 전설을 내 손으로 재현하는 역사적 몰입 | Supporting |

### Player Type Appeal (Bartle Taxonomy)

- [x] **Achievers** — 보스 처치, 챕터 완료, 혈기 최대 비축 달성
- [ ] **Explorers** — 세계관 탐색은 부차적 (미니멀 스토리텔링)
- [ ] **Socializers** — 싱글플레이어
- [x] **Killers/Competitors** — 퍼펙트 클리어, 최소 피격, 최대 혈기 흡수 도전

### Flow State Design

- **온보딩 커브**: 1챕터 초반 — 적 소수, 혈흔 밀집, 흡수 효과 시각적으로 명확하게. 플레이어가 설명 없이 혈흡수를 발견하도록 유도.
- **난이도 스케일링**: 챕터 진행 → 적 밀도 증가, 엘리트 패턴 복잡화, 혈기 소비 압박 증가.
- **피드백 명확성**: 혈기 게이지 시각화, 흡수 완료 시 음향+비주얼 피드백, 보스 페이즈 전환 시 혈기 방출 효과.
- **실패 회복**: 챕터 내 체크포인트. 죽으면 직전 체크포인트로 복귀 (혈기 일부 유지 또는 리셋 — 설계 검토 필요).

## Core Loop

### Moment-to-Moment (30초)

```
적 배치 읽기 → 유리한 위치 선점 → 범위/방향 기술로 군중 제압
→ 적 처치 → 혈흔 생성 → [판단] 지금 흡수? 잔적 먼저?
→ 흡수 발동 (취약 구간) → 혈기 충전 → 다음 교전
```

포지셔닝과 흡수 타이밍이 매 30초 루프의 핵심 결정이다.

### Short-Term (5~15분)

```
구역 입장 → 잡몹 클리어 (혈기 비축) → 엘리트 적 발견
→ [전략 결정] 지금 혈기 쓸까, 아끼고 다음 구역으로?
→ 구역 보상 획득 → 다음 구역
```

"혈기 지출 타이밍"이 구역마다 반복되는 긴장의 원천.

### Session-Level (30~90분)

```
챕터 시작 → 2~3 구역 탐색 (혈기 비축 단계)
→ 챕터 보스 조우 → 비축 혈기 전략적 방출 → 보스 처치
→ 영구 강화/스킬 언락 → 다음 챕터 개방
```

보스전이 챕터 전체의 혈기 관리 결산점이다.

### Long-Term Progression

- 챕터 클리어 시 새 스킬 또는 무기 변형 언락
- 각 챕터 적 유형의 혈기 효과 완전 해금
- 소야차의 전설 서사 완결 (3챕터)
- 챌린지 모드: 최소 혈기 사용, 무피격 클리어 등

### Retention Hooks

- **호기심**: 다음 챕터의 적 유형과 혈기 효과 미리 공개되지 않음. "요괴의 피는 어떤 힘을 줄까?"
- **투자**: 언락된 스킬과 캐릭터 강화. 챕터 클리어 기록.
- **숙련**: 전 구역을 더 적은 혈기로 클리어하는 효율 추구. 퍼펙트 흡수 달성.

## Game Pillars

### Pillar 1: 읽고 치는 자

모든 전투는 포지셔닝에서 시작한다. 적 배치를 읽고 유리한 자리를 만든 뒤 공격한다. 소야차는 무모하지 않다 — 이미 이긴 싸움만 한다.

*Design test*: 자동조준을 추가할까, 수동 조준을 유지할까? → 이 기둥이 "수동 조준"을 선택한다. 위치 선정의 의미가 살아야 한다.

### Pillar 2: 피를 지배하는 자

전장의 피는 소야차의 자원이다. 흘린 피를 직접 흡수해야만 충전된다. 언제, 어디서 흡수하느냐가 전략이다. 혈기는 항상 부족하게 설계한다.

*Design test*: 자동 흡수 기능을 추가할까? → 이 기둥이 "기각"한다. 흡수 결정이 플레이어의 것이어야 한다.

### Pillar 3: 조선의 공포

세계관, 적 설계, 내러티브 모두 조선 역사와 전설에 뿌리를 둔다. 소야차의 실존 전설이 메커닉의 원천이다. 장르 판타지를 더하되 조선이라는 맥락을 잃지 않는다.

*Design test*: 이 적/스킬/장소가 조선을 배경으로 설명 가능한가? → 불가능하면 재검토. 이무기는 가능, 서양 드래곤은 기각.

### Anti-Pillars (What This Game Is NOT)

- **무쌍 게임이 아니다**: 버튼 연타로 이기는 게임이 아님. 포지셔닝 없이 강타만으로는 죽는다. 무쌍형 카타르시스는 기둥 1을 파괴한다.
- **로그라이크가 아니다**: 죽으면 처음부터 다시하는 구조 없음. 챕터 기반 선형 진행, 영구 성장. 런 기반 구조는 혈기 비축 전략의 의미를 희석시킨다.
- **오픈월드가 아니다**: 6개월 안에 오픈월드는 없다. 밀도 높은 선형 챕터로 집중. 탐험보다 전투 설계에 자원을 쓴다.
- **공포 게임이 아니다**: 소야차는 공포를 주는 쪽이다, 겁먹는 쪽이 아니다. 다크하되 무겁지 않다. 공포 연출에 집중하면 판타지 기둥이 흔들린다.

## Risks and Open Questions

### Design Risks

- **혈흡수가 "버프 게이지"로 느껴질 위험** — 흡수의 시각적/청각적 연출이 충분히 임팩트 있지 않으면 전략 자원이 아닌 단순 게이지로 인식될 수 있다.
- **포지셔닝의 강제성 부재** — 맵 설계가 허술하면 포지셔닝 없이도 클리어 가능해진다. 레벨 디자인이 기둥 1을 뒷받침해야 한다.
- **보스전 직전 혈기 소진 시 난이도 스파이크** — 혈기를 다 써버리고 보스를 만나는 경우 극단적 난이도 상승. 패널티 vs. 설계 의도 조율 필요.

### Technical Risks

- **혈흔 파티클 성능** — 대형 전투 후 다수 혈흔이 동시에 흡수될 때 파티클 수가 폭증. 성능 예산 확보 필요.
- **흡수 이펙트 품질** — "피가 공중을 날아와 주인공에게 빨려 들어가는" 비주얼이 기대치에 미치지 못할 경우 핵심 차별점이 희석됨.

### Market Risks

- **조선 배경의 생소함** — 글로벌 시장에서 낯선 배경은 진입 장벽이 될 수 있음. (반대로 차별화 포인트이기도 함.)
- **인디 ARPG 시장 경쟁** — 하데스 이후 이소메트릭 인디 액션이 증가. 차별점 명확히 해야 함.

### Scope Risks

- **레벨 디자인 공수 과소평가** — 포지셔닝이 의미 있으려면 각 구역의 맵 설계에 상당한 공수 필요. 6개월 일정에서 가장 큰 병목.
- **적 유형 설계 볼륨** — 챕터당 3~4종, 총 9~12종. 각 유형별 행동 패턴 + 혈기 효과 설계가 예상보다 많은 이터레이션 필요.

### Open Questions

- **흡수 중 취약 구간의 길이** — 얼마나 길어야 "위험하지만 가치 있는" 결정이 될까? 프로토타입으로 0.5초 / 1초 / 1.5초 테스트 필요.
- **혈기 소비 시 사망 패널티** — 보스전 시작 시 혈기를 얼마나 유지시킬까? 체크포인트에서 전량 리셋 vs. 일부 유지 vs. 유지 (프로토타입 플레이테스트로 결정).
- **혈기 종류 수** — 챕터별 1종(3종 총합) vs. 적 유형별 상이한 효과. 6개월 내 구현 가능한 수준 vs. 전략적 깊이 사이 균형.

## MVP Definition

**Core hypothesis**: 포지셔닝 전투 + 혈흡수 타이밍 결정이 30분 이상 세션에서 플레이어를 몰입시키는가?

**Required for MVP**:
1. 혈흡수 시스템 — 혈흔 생성, 흡수 발동, 혈기 충전, 방출 (비주얼/사운드 포함)
2. 포지셔닝 전투 — 방향/범위 기술, 회피, 카타나 콤보
3. 1챕터 완성 — 3~4 구역 + 보스 1명. 핵심 루프 검증용.
4. 혈기 종류 최소 1종 (효과 명확히)
5. 진행 저장 및 체크포인트

**Explicitly NOT in MVP**:
- 2챕터, 3챕터 (루프 검증 후 추가)
- 활 무기 시스템 (카타나로 MVP 루프 집중)
- 영구 업그레이드 트리 (기본 성장만)
- 스토리 연출/컷씬
- 적 혈기 종류 다양화

### Scope Tiers

| Tier | Content | Features | Timeline |
| ---- | ---- | ---- | ---- |
| **MVP** | 1챕터, 보스 1명 | 혈흡수 + 포지셔닝 전투 + 카타나 | 8주 |
| **Vertical Slice** | 1챕터 완성도 높게 | 혈기 1종 + 활 추가 + 영구 강화 기초 | 12주 |
| **Alpha** | 3챕터 (미완성) | 혈기 3종 + 스킬 트리 초안 + 보스 3명 | 20주 |
| **Full Vision** | 3챕터 완성 + 챌린지 모드 | 전체 기능 + 연출 + 밸런싱 | 24주 |
